# Travo
Smart Tourism Companion for Rwanda 🌍

##  Overview
Travo is a tourism web application designed to showcase Rwanda’s top destinations.  
It provides visitors with destination pages, booking forms, and an admin dashboard to manage site activity.

## ✨ Features
-  Destination pages for Akagera, Nyungwe, Lake Kivu, and Volcanoes National Park
-  Booking forms with user-friendly design
-  Login page for secure access
-  Admin dashboard to monitor bookings and site activity
-  Integration of Rwanda-specific tourism data
